# Mímica Pro

Juego de mímicas divertido para toda la familia.

## Cómo jugar
1. Selecciona una o más categorías.
2. Presiona "EMPEZAR".
3. Actúa la palabra que aparece en pantalla sin hablar.
4. ¡Adivina tantas como puedas antes de que se acabe el tiempo!

## Instalación (PWA)
Puedes instalar esta aplicación en tu celular abriendo el link en Chrome (Android) o Safari (iOS) y seleccionando "Agregar a la pantalla de inicio".
